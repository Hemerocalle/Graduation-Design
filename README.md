# 毕业设计

## 题目
> 基于机器视觉的人脸表情识别系统设计

- 题目性质: 计算机软件型
- 题目类型: 毕业设计
- 题目来源: 科研课题

## 主要内容	
> 识别人脸表征表情的开心、伤心、惊讶、愤怒等多种表情。 

## 基本要求
> 理工类毕业设计要求学生必须基于专业知识，
> 对本课题要解决的问题在社会环境下进行合理分析，
> 评价对社会、健康、安全、文化以及可持续发展的影响，
> 以及不同方案的优缺点和经济成本分析

1. 完成人脸表征表情的主要特征的分割;
2. 实现表情识别;
3. 能够基于工程相关背景知识进行合理分析，评价专业工程实践和复杂工程问题解决方案对社会、健康、安全、法律以及文化的影响，并理解应承担的责任。
